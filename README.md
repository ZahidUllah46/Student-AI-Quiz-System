# Student AI Quiz System

A professional and interactive Artificial Intelligence Quiz Application built using HTML, CSS, and Vanilla JavaScript.

This project was developed as part of a Web Development Internship Assignment to demonstrate front-end development skills, JavaScript logic building, DOM manipulation, timer implementation, score tracking, and responsive UI design.

---

## Project Overview

Student AI Quiz System is a browser-based quiz application that allows students to enter their details and attempt an Artificial Intelligence assessment.

The application includes a real-time countdown timer, automatic question progression, answer validation, score calculation, performance evaluation, and a professional results dashboard.

---

## Features

### Student Registration

* Student Name validation
* Roll Number validation
* Professional welcome screen

### Quiz Functionality

* 10 Artificial Intelligence MCQs
* One question displayed at a time
* Four answer options per question
* Correct answer highlighting
* Incorrect answer highlighting
* Automatic option disabling after selection

### Timer System

* 30-second countdown for every question
* Automatic timer reset for each question
* Auto-skip when time expires
* Automatic navigation to next question

### Progress Tracking

* Question counter
* Percentage completion
* Animated progress bar

### Result Dashboard

* Student Name
* Roll Number
* Correct Answers
* Wrong Answers
* Skipped Questions
* Final Score
* Percentage
* Performance Evaluation

### Restart Functionality

* Restart Quiz
* Reset Score
* Reset Timer
* Shuffle Questions
* Return to Welcome Screen

### User Interface

* Modern Design
* Responsive Layout
* Mobile Friendly
* Professional Educational Theme
* Smooth Animations
* Clean Typography

## How to Run

### Method 1: Open Directly

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.
4. Enter Student Name and Roll Number.
5. Start the quiz.

### Method 2: VS Code Live Server

1. Open project in VS Code.
2. Install Live Server Extension.
3. Right-click on `index.html`.
4. Click "Open with Live Server".
